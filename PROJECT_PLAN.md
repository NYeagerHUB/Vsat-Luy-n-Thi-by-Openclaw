# Project Plan — V-SAT Luyện Thi

## Mục tiêu
Web luyện thi V-SAT & THPT online, vibe giáo dục sạch, hiện đại, inspired by thitrenmaytinh.edu.vn.

## Style reference
- Nền trắng + gradient xanh dương nhạt ở hero
- Primary: `#2563eb`
- Card bo góc 16px, shadow nhẹ
- Section spacing lớn (96px)
- Hero 2 cột: text trái, mock exam phải
- Responsive: mobile first breakpoint ở 768px

## Cấu trúc trang hiện tại
1. Header — logo + nav + CTA
2. Hero — title + desc + 2 nút + mock exam card
3. Features — 4 card grid
4. Formats — 2 card (V-SAT + THPT)
5. CTA cuối trang
6. Footer
